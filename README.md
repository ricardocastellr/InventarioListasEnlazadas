# Inventario Almacen
Práctica de alamcen CRUD:
Se va a controlar un inventario de productos, de los que se guarda:
- Código.
- Nombre.
- Cantidad.
- Costo.

Utilizando vectores y podremos:
- Agregar.
- Buscar por codigo. 
- Eliminar por codigo. 
- Insertar un producto en una posicion.
- Recuperar el listado como texto.
